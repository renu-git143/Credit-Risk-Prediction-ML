# Credit Risk Prediction using Machine Learning

## Overview
This project predicts customer credit risk for a banking institution using machine learning models.

## Tools & Technologies
- Python, Pandas, NumPy
- Scikit-learn, XGBoost
- MySQL
- Matplotlib, Seaborn

## Steps Performed
- Data extraction from MySQL
- Data preprocessing and encoding
- Exploratory Data Analysis (EDA)
- Feature engineering
- Machine learning models: Logistic Regression, Random Forest, XGBoost, KNN, SVM
- Evaluation using ROC-AUC, Gini coefficient, Decile Ranking
